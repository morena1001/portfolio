About

I am Josué Flores, a graduate with a bachelor's degree in Computer Science, and passionate about web, game, and software development. Throughout my college career, I studied topics covering mathematics like algorithms and data structures, to theoretical topics like discrete math and computability, to more hands-on topics like databases, operating systems, deep learning, and assembly language.

Curently, my main focus is web development using HTML, JavaScript, and some lightweight Node.js tools like JSON server. I have also recently retaken interest in programming with assembly language and game development using Unity, as well as taken a new interest in electrical engineering to code electronics using a Raspberry Pi.

Proficient in C, C++, C#, HTML and JavaScript, and Java. Familiar with Python, SQLitem Pytorch, Dart and Flutter.
